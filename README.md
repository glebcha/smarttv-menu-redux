TV App
==========
![tvapp](https://infinit.io/_/9NDLfFb.jpg)

###How to run:
1 - Execute 'npm start'
	to run in development mode (open localhost:3000).

2 - Execute 'npm test' to run tests.

3 - Execute 'npm run build' to build app.

----------


> **What's in the box:**

> - React + Redux.
> - Karma/Mocha.
> - Webpack.
